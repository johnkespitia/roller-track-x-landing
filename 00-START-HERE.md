# Roller Track X — Paquete de contexto para Cursor (Web MVP)

Este folder contiene la especificación para que Cursor construya la web MVP de Roller Track X con enfoque low-cost.

## Entregables esperados
1) Landing principal (Home) con secciones: propuesta de valor, problema, solución, piloto, audiencias, CTA.
2) Captación (leads) con formularios para: Escuelas/Clubes, Deportistas/Familias y Sponsors.
3) Medición: analítica básica (eventos de clicks CTA y envíos de formulario).

## Principios
- No prometer IA avanzada ni ranking oficial en el MVP; todo es piloto y seguimiento manual.
- Velocidad: performance, mobile-first, SEO básico.
- Branding: RTX (Roller Track X), tagline: The Next Revolution of Speed.

- PRD.md (producto y alcance)
